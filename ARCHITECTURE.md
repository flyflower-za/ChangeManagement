## 核心业务流程

### 1. 变更创建流程
```
用户创建变更
    │
    ▼
填写表单 (标题、描述、优先级、时间)
    │
    ▼
选择涉及的模块
    │
    ▼
POST /api/changes
    │
    ├─ 创建 ChangeProject (状态: DRAFT → PENDING)
    │
    ├─ 对每个选中的模块:
    │   │
    │   ├─ 获取模块的 ChecklistTemplate
    │   │
    │   ├─ 创建 ChangeModule (审批人 = 模块负责人)
    │   │
    │   └─ 根据 Template 实例化 ChecklistItem
    │
    └─ 更新状态为 EXECUTING
        │
        ▼
跳转到变更详情页
```

### 2. Checklist 执行流程
```
执行人查看任务 (/my-tasks)
    │
    ▼
选择待执行的 ChecklistItem
    │
    ▼
填写执行证据
    │
    ▼
PATCH /api/changes/[id] (action='execute_item')
    │
    ├─ 更新 ChecklistItem 状态为 DONE
    │
    ├─ 检查模块内所有 Item 是否完成
    │   │
    │   └─ 是 → 更新 ChangeModule 状态为 REVIEWING
    │
    └─ 检查所有模块是否都在 REVIEWING
        │
        └─ 是 → 更新 ChangeProject 状态为 APPROVING
```

### 3. 审批流程
```
模块负责人查看待审批任务
    │
    ▼
审批操作 (/approvals 或变更详情页)
    │
    ├─ 通过: PATCH (action='approve_module')
    │   │
    │   ├─ 更新 ChangeModule 状态为 APPROVED
    │   │
    │   └─ 检查所有模块是否都已 APPROVED
    │       │
    │       └─ 是 → 更新 ChangeProject 状态为 COMPLETED
    │
    └─ 驳回: PATCH (action='reject_module')
        │
        ├─ 可选择驳回特定 Item 或全部 Item
        │
        ├─ 更新相关 Item 状态为 REJECTED
        │
        ├─ 更新 ChangeModule 状态回 EXECUTING
        │
        └─ 更新 ChangeProject 状态回 EXECUTING
```

## 状态机模型

### ChangeProject 状态流转
```
     ┌─────────┐
     │ DRAFT   │ (草稿)
     └────┬────┘
          │ 提交
          ▼
    ┌─────────┐
    │ PENDING │ (待执行)
    └────┬────┘
         │ 有模块开始执行
         ▼
   ┌──────────┐
   │ EXECUTING│ (执行中)
   └────┬─────┘
        │ 所有模块完成执行
        ▼
   ┌──────────┐
   │ APPROVING│ (待审批)
   └────┬─────┘
        │ 所有模块审批通过
        ▼
   ┌──────────┐
   │COMPLETED │ (已完成)
   └──────────┘

任何状态 ──取消──> CANCELLED
```

### ChangeModule 状态流转
```
┌──────────┐
│ PENDING  │ (待执行)
└────┬─────┘
     │ 开始执行
     ▼
┌──────────┐
│ EXECUTING│ (执行中)
└────┬─────┘
     │ 所有 Item 完成
     ▼
┌──────────┐
│REVIEWING │ (待审批)
└────┬─────┘
     │       │
   通过    驳回
     │       │
     ▼       ▼
┌──────┐ ┌──────────┐
│APPROVED│ │ REJECTED │
└──────┘ └─────┬────┘
               │
               │ 需重做
               ▼
          ┌──────────┐
          │ EXECUTING│
          └──────────┘
```

### ChecklistItem 状态流转
```
┌─────────┐
│ PENDING │ (待执行)
└────┬────┘
     │ 开始执行
     ▼
┌────────────┐
│ IN_PROGRESS│ (执行中)
└────┬───────┘
     │ 完成提交
     ▼
┌────────┐     ┌─────────┐
│  DONE  │<───>| REJECTED│
└────────┘     └────┬────┘
                     │
                     │ 需重做
                     ▼
                ┌────────────┐
                │ IN_PROGRESS │
                └────────────┘
```
