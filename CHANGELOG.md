# Changelog

## 2026-06-27 - 变更管理系统全面升级

### 数据模型重构
- **Factory** 工厂实体（可扩展多工厂）
- **Module** 部门/车间平级，关联工厂
- **Product** 产品独立实体，跨部门负责人关联
- **ProductAssignment** 产品-部门-负责人（多人多部门）
- **ProductionLine** 产线实体（属于部门、关联产品）
- **DepartmentContact** 部门联系人
- **EmailTemplate** 邮件模板（可自定义HTML）
- **PermissionGroup** 权限组（关联系统角色）
- **LdapConfig** LDAP 目录服务配置
- **AiConfig** AI 审批助手配置
- **SmtpConfig** SMTP 邮件配置（加密方式可选）
- **Attachment** 检查项附件（图片/PDF/Word/Excel）

### 变更项目
- 自增编号 `#1`, `#2`...
- 产品关联，创建变更时自动带入各部门负责人
- 删除/归档功能（仅管理员和发起人）
- NOT_APPLICABLE 检查项状态
- 已完成变更锁定（仅admin可编辑）
- 搜索：编号、优先级、标题、产品、状态、发起人

### 检查项执行
- 表格列表样式，紧凑单行
- 文件上传支持（图片/PDF/Word/Excel）
- 批量分配执行人
- 产品-部门负责人自动匹配显示
- 单项驳回 + 批量驳回

### 审批体验
- 审批中心：每人只看自己部门的待审批模块
- 筛选器：全部/已完成/不涉及/已驳回/待执行
- 单项驳回 ↺ + 批量驳回
- 审批通过二次确认弹窗
- AI 审批摘要（配置后可用）
- 执行与审批合并为一个检查项标签

### 管理中心（5标签）
- 🏢 部门管理 · 📦 产品管理 · 📝 模版管理 · 📧 邮件配置 · 🤖 AI配置

### 邮件通知（8种）
- 创建变更 → 发起人+审批人
- 检查项完成 → 审批人
- 审批通过/驳回 → 发起人
- 截止7天/3天提醒
- 检查项驳回 → 执行人
- 检查项分配 → 执行人
- 变更完成 → 全员
- 每日/每周汇总

### 用户管理（3标签）
- 👥 用户列表（含部门/权限组/角色）
- 🔐 权限组管理
- 📡 LDAP 配置（含填写帮助）

### UI 优化
- 变更项目紧贴表格列表
- 仪表盘同步列表样式
- 变更历史同步列表样式
- 我的待办表格化
- 变更详情两个标签：概览 + 检查项
